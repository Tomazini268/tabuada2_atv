Gerador de Tabuadas com While

Sobre o Projeto

Este projeto é um Gerador de Tabuadas desenvolvido com HTML, CSS e JavaScript, utilizando a estrutura de repetição while para gerar automaticamente as tabuadas.

Ao clicar no botão "Gerar Todas as Tabuadas", o sistema exibe as tabuadas organizadas na tela.

Funcionalidades

Geração automática de múltiplas tabuadas

Uso da estrutura de repetição while

Organização das tabuadas em caixas na tela

Layout simples e organizado

Manipulação do DOM com JavaScript

Tecnologias Utilizadas

HTML

CSS

JavaScript
